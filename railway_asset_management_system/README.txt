
RAILWAY ASSET MANAGEMENT SYSTEM

TECH STACK
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MySQL

HOW TO RUN

1. Install Node.js
2. Open terminal in backend folder

Install packages:
npm install express mysql2 cors

Run backend:
node server.js

3. Setup MySQL
- Open MySQL
- Run schema.sql

4. Open frontend/index.html in browser
